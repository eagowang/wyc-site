## 作用域

es6 之前，js 只有全局作用域和函数作用域

```js
var a = 1;
function add() {
  a = a + 1;
  console.log(a); //2
}
add();
```

```js
var a = 1;
function add() {
  var a = a + 1; //NaN
  console.log(a);
}
add();
```
